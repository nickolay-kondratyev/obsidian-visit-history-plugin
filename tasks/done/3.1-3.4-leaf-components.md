> **Depends on:** Tasks 2.1, 2.2

## Common Notes for All Four Components

### Preact+htm → React+JSX Conversion

| Prototype | Plugin |
|-----------|--------|
| `html\`<div className=${x}>\`` | `<div className={x}>` |
| `html\`<${Comp} prop=${val}/>\`` | `<Comp prop={val} />` |
| `html\`<${Fragment}>...<//>\`` | `<>...</>` |
| `style=${{ color:'red' }}` | `style={{ color: 'red' }}` |
| `key=${"x"+i}` | `key={"x" + i}` |

### SVG CamelCase (for FolderNode, LeafNode)

| SVG attr | JSX |
|----------|-----|
| `font-family` | `fontFamily` |
| `font-size` | `fontSize` |
| `font-weight` | `fontWeight` |
| `letter-spacing` | `letterSpacing` |
| `fill-opacity` | `fillOpacity` |
| `stroke-width` | `strokeWidth` |

### Imports Pattern

All components import from `'react'` (not preact):
```typescript
import { useState, useEffect, useRef, useMemo, Fragment } from 'react';
```

Since we use `"jsx": "react-jsx"`, no `import React` needed.

## 3.1 — Legend.tsx

**Props:** `{ colorMode: 'type' | 'heatmap', gradKey: string }`

Pure presentational. No state. No Obsidian API.

- When `colorMode === 'type'`: render colored pills for md/canvas/excalidraw.
- When `colorMode === 'heatmap'`: render gradient bar + "no data" indicator.

## 3.2 — Tooltip.tsx

**Props:** `{ x: number, y: number, node: HierarchyNode<VaultNode>, scales: Record<string, number> }`

Pure presentational. No state.

- Shows: name, type badge, path, raw size, effective size, 3 timestamps.
- Null timestamps rendered as "—" with italic style.

## 3.3 — FolderNode.tsx

**Props:** `{ d: HierarchyNode<VaultNode> }`

Pure SVG. No state, no event handlers.

- Renders a `<rect>` for the folder with depth-based fill color.
- Renders `<text>` label if height > 14px.
- Uses nested `<svg overflow="hidden">` for clipping (same as prototype).

## 3.4 — LeafNode.tsx

**Props:**
```typescript
{
  d: HierarchyNode<VaultNode>;
  hovered: boolean;
  colorMode: 'type' | 'heatmap';
  gradKey: string;
  field: string;
  hotDays: number;
  coldDays: number;
  onMouseMove: (e: React.MouseEvent, d: HierarchyNode<VaultNode>, i: number) => void;
  onMouseLeave: () => void;
}
```

- Renders a `<rect>` with fill from `leafFill()` and opacity from `leafOpacity()`.
- Renders `<text>` filename if cell is big enough (w > 32, h > 14).
- Hover state is owned by parent (TreemapViz) — this component just receives `hovered`.

**Future:** Add `onClick` to open the file in Obsidian (not part of this migration step).
