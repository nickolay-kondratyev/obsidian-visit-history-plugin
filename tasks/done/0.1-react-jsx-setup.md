# Task 0.1: Add React/JSX Support to Build Toolchain

> **Phase:** 0 — Infrastructure
> **Depends on:** nothing
> **Blocks:** all other tasks

## Steps

### 1. tsconfig.json — enable JSX

Add to `compilerOptions`:
```json
"jsx": "react-jsx"
```

Using `react-jsx` (not `react`) means no need to `import React from 'react'` in every file — the JSX transform auto-imports `jsx` from `react/jsx-runtime`.

### 2. esbuild.config.mjs — mark React as external

Add `'react'` and `'react-dom'` to the `external` array. Obsidian bundles React 18 internally; we must NOT bundle our own copy.

### 3. package.json — add dependencies

```bash
npm install --save d3-hierarchy d3-interpolate d3-color @visx/zoom
npm install --save-dev @types/react @types/react-dom @types/d3-hierarchy @types/d3-interpolate @types/d3-color
```

### 4. Verify

Run `npm run build` — should succeed with no `.tsx` files yet.

Create a minimal smoke-test file to confirm JSX compiles:
```tsx
// src/view/__smoke__/HelloWorld.tsx
export function HelloWorld() {
  return <div>Hello from React</div>;
}
```

Run `npm run build` again — should succeed.
Then delete the smoke-test file (or keep it as a dev-only sanity check).
